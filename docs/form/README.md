# 基础用法

从这一节开始，我们将逐个的介绍表单域的基础使用方法

所有的表单域都有如下的属性

## name

每个表单域都有一个 `name` 属性，而且不能为空，通过这个值与 `form` 进行交互

## id

每个表单域都有一个 `id` 属性，会赋值到最终显示在 `html` 中的 `id` 属性值。
如果不设置，组件会自动生成一个前缀为 `input_` 的随机字符串

## value

每个表单域都可以设置 `value` 属性，对应表单域的值，这在你的编辑表单中会很有用处
